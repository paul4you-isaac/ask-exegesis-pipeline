# 본문주해 파이프라인 셋업 (ask-exegesis-setup)

ask-exegesis 1~8단계 스킬이 작동하려면 프로젝트 폴더(옵시디언 볼트)에 ① 파이프라인 지침,
② `주해작업/` 폴더, ③ 설교자 프로필이 있어야 한다. 이 스킬은 그 세 가지를 인터뷰로 만든다.

## 실행 조건 판별 (항상 먼저)

1. 프로젝트 폴더 루트의 `CLAUDE.md`에 `## 본문주해 8단계 파이프라인` 섹션이 있는지,
   `주해작업/00_설교자프로필.md`가 있는지 확인한다.
2. **둘 다 있으면**: "이미 셋업되어 있습니다"라고 알리고, 프로필 요약을 보여 준 뒤
   수정을 원하는지 묻는다. 수정 요청이 없으면 종료하고 1단계(또는 진행 중 단계)를 안내한다.
3. **없으면**: 아래 인터뷰를 진행한다.
4. 프로젝트 폴더가 연결되지 않은 환경이면, 옵시디언 볼트(또는 작업 폴더) 연결을 먼저 청한다.

## 인터뷰 (한 번에 한 질문, 따뜻한 경어체)

첫 메시지에 이미 담긴 정보는 다시 묻지 않는다. 각 항목은 **예시를 제공하여** 설교자가
고르거나 자유롭게 답하게 한다. 호칭은 직분 확인 전까지 "설교자님"을 쓴다.

### ① 이름과 호칭
"먼저 성함(또는 원하시는 호칭)을 알려 주시겠습니까?"

### ② 교회(사역지)
"섬기시는 교회(또는 사역지) 이름을 알려 주시겠습니까?" — 신학생 등 소속 교회가 없으면
"소속 없음"도 가능하다고 안내한다.

### ③ 직분
예시 제공: 담임목사 / 부목사 / 전도사·강도사 / 선교사 / 신학생 / 평신도 설교자 / 기타.
확정된 직분에 따라 이후 전 과정의 호칭을 정한다(예: 목사님, 전도사님, 선교사님,
전도사·강도사·신학생·평신도는 본인이 원하는 호칭을 물어 따른다).

### ④ 신학 좌표
예시를 제공하며 묻는다: "설교와 해석의 기준으로 삼으시는 신학 전통은 어디에 가깝습니까?"
- 개혁주의·장로교 (웨스트민스터 신앙고백)
- 침례교 (침례교 신앙고백)
- 웨슬리안·감리교
- 오순절·순복음
- 루터교 (아우크스부르크 신앙고백)
- 복음주의 일반 / 기타 (직접 서술)

답을 받으면 한 줄로 확인한다. 이 좌표는 지침의 「해석 좌표」에 기록되어, 신학 판단이
갈리는 지점에서 기본값이 된다. 단, 김희석 방법론의 공통 기반(역사적·문법적·신학적 해석,
구원 계시의 점진적 발전)은 전통과 무관하게 파이프라인의 뼈대임을 안내한다.

### ⑤ 설교 철학
예시를 제공하며 묻는다: "설교를 준비하실 때 지향하시는 방향은 무엇입니까?"
- 그리스도 중심 강해설교 (구속사적 궤적, Bryan Chapell의 FCF)
- 본문 중심 강해설교 (본문의 구조와 메시지를 따라)
- 주제·시리즈 설교 병행
- 청중 중심·삶의 적용 강조
- 기타 (직접 서술)
복수 선택도 허용한다.

### ⑥ 주요 사역 인터뷰 (자유 대화)
정해진 보기 없이 짧게 인터뷰한다. 다음을 자연스럽게 묻고, 답에 따라 한두 번 더 깊이 묻는다:
- "주일 설교는 몇 회나 하십니까? 새벽·수요·금요 등 다른 설교 사역도 있으신가요?"
- "설교 준비에 보통 며칠을 쓰십니까? 옵시디언(또는 마크다운 편집기)은 어느 정도
  익숙하십니까?"
- "설교 외에 큐티·양육·심방 등 이 주해 작업과 연결하고 싶은 사역이 있습니까?"
답을 요지 그대로 프로필에 기록한다(Claude의 해석을 덧붙이지 않는다).

## 설치 (인터뷰 완료 후)

1. **프로필 저장**: `주해작업/00_설교자프로필.md` 생성 —
   YAML(이름, 호칭, 교회, 직분, 신학좌표, 설교철학, 작성일) + 「주요 사역」 본문(인터뷰 요지).
2. **지침 설치**: 함께 제공되는 `references/project-instructions-template.md`를 읽어
   `{{플레이스홀더}}`를 인터뷰 결과로 치환한 뒤:
   - 프로젝트 폴더에 `CLAUDE.md`가 없으면 그 내용으로 새로 만든다.
   - 이미 있으면 맨 끝에 `## 본문주해 8단계 파이프라인` 섹션으로 추가한다(기존 내용은
     건드리지 않는다).
3. **폴더 생성**: `주해작업/` 폴더가 없으면 만든다.
4. **완료 안내**: 설치 내역을 두세 줄로 요약하고, "이제 '주해 시작' 또는 '1단계'라고
   말씀하시면 본문 확정부터 시작합니다"라고 안내한다. 이후 모든 세션에서 설치된 지침과
   프로필의 호칭·신학 좌표를 따른다.

## 규칙

- 인터뷰는 판단하거나 평가하지 않는다. 어떤 전통·철학이든 존중하며 기록한다.
- 프로필·지침에 설교자가 말하지 않은 내용을 추가하지 않는다.
- 이미 설치된 지침을 덮어쓰지 않는다(수정 요청 시에만 해당 항목을 갱신).
- 프로필 수정 요청("프로필 수정", "신학 좌표 바꿔줘")이 오면 해당 항목만 다시 묻고
  프로필 파일과 지침의 관련 구절을 함께 갱신한다.
