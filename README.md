# ask-exegesis-pipeline

김희석 「성경으로 설교하기」의 본문주해 8단계를 Claude(Cowork)와 옵시디언에서
그대로 밟아 가는 설교 준비 파이프라인 플러그인입니다. Claude는 답을 대신 쓰는
도구가 아니라, 소크라테스식 질문으로 설교자가 스스로 주해하도록 돕는 **코치**입니다.

## 구성 (스킬 9개)

| 스킬 | 역할 |
|------|------|
| ask-exegesis-setup | **첫 실행 셋업** — 이름·교회·직분·신학 좌표·설교 철학·주요 사역 인터뷰 후 프로젝트 지침 자동 설치 |
| ask-exegesis-1-reading-v02 | 1단계 본문 확정과 개인 번역(사역) |
| ask-exegesis-2-context-v02 | 2단계 역사적 정황 연구 |
| ask-exegesis-3-words-v02 | 3단계 단어 연구 |
| ask-exegesis-4-genre-v02 | 4단계 장르 연구 |
| ask-exegesis-5-plot-v02 | 5단계 문학적 정황(문맥·구조·주제 흐름) |
| ask-exegesis-6-canon-v02 | 6단계 정경적 정황(구약·신약 비교, 정경 흐름) |
| ask-exegesis-7-theology-v02 | 7단계 신학과 메시지(의미의 보편화) |
| ask-exegesis-8-outline-v02 | 8단계 적용 포인트와 설교 개요 |

## 설치

1. [Releases](../../releases) 또는 저장소에서 `ask-exegesis-pipeline.plugin` 파일을 내려받습니다.
2. Claude 데스크톱 앱(Cowork)에서 파일을 열면 설치 버튼이 나타납니다.
   (또는 설정 → Capabilities에서 플러그인으로 추가)

## 시작하기

1. Cowork에서 **옵시디언 볼트(작업 폴더)를 연결**합니다.
2. "주해 셋업" 또는 "본문주해 시작"이라고 입력합니다.
3. 첫 실행이면 셋업 인터뷰가 진행되고, 폴더에 파이프라인 지침(CLAUDE.md)·
   `주해작업/` 폴더·설교자 프로필이 만들어집니다.
4. 이후 "1단계"부터 시작해 여덟 단계를 밟습니다. 어느 단계에서든
   "전체 현황" / "이어서" / "저장" / "작성완료" 명령을 쓸 수 있습니다.

## 작동 방식

- 산출물은 `주해작업/주해N_[책명]_[장,절-절]_[날짜]_[접미어].md`로 저장되며,
  각 파일 끝의 **인계 블록**을 다음 단계가 자동으로 읽어 이어 갑니다.
- 5~8단계의 핵심 산출(주제 흐름·정경 흐름·메시지 명제·설교 개요)은 Claude가
  뼈대만 기록하고, **설교자가 옵시디언에서 직접 채운 뒤 "작성완료"**로 점검받습니다.
- 성경 본문(새한글·개역개정·새번역·공동번역, BHS/NT28)은 저작권 보호를 위해
  **설교자가 직접 붙여넣은 텍스트만** 사용합니다. Claude가 생성하지 않습니다.

## 크레딧

- 방법론: 김희석, 「성경으로 설교하기」 (본문주해 8단계) — 본 플러그인은 그 과정을
  코칭 워크플로우로 구현한 것이며, 책의 본문을 포함하지 않습니다. 책과 함께 사용하기를
  권합니다.
